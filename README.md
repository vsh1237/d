# Bento OS Final

This repository contains the source for a custom HTML dashboard named **Bento OS Final**. The project is a single-page application defined in `index.html` that emulates an operating system-like interface with various widgets and utilities.

## Features

- Dynamic clock, date, and system statistics (battery, network, OS).
- Command center with search shortcuts and quick links (Mail, AI, YouTube).
- Habit tracker with persistence controls.
- Environment module showing moon phase and location.
- News ticker with scrolling headlines.
- Agenda view powered by a Google Calendar embed.
- Priority task list with progress bar, priority selector, and management.
- Quick launch dock with links to popular services.
- Pomodoro-style focus timer with selectable sounds.
- Scratchpad supporting markdown preview, text-to-speech, and export.
- A drawer containing Tabs for Utilities, Dev Tools, Wellness, and System.
- Mini-tools inside drawer: QR generator, WPM test, password generator, calculator, dice & coin.
- Screensaver that activates on idle with time and quotes.
- Theme toggling between light and dark modes via `data-theme` attribute.
- Responsive grid layout using CSS Grid with glassmorphic styling.

## Usage

Open `index.html` in a modern browser to run the dashboard locally. The page uses built-in JavaScript for interactivity and local storage for persistence of habits, tasks, etc.

## Development

This is a static HTML/CSS/JS project; no build steps are required. Modify `index.html` directly to change the layout or functionality. Ensure any changes are tested in modern browsers (Chrome, Firefox, Edge).
